# TimeWarAssets
The time war game assets

بازی time war یه بازیه در حال ساخته،شما هم میتونین در ساخت شخصیت ها و صداها نقش داشته باشید
چطوری؟

## برای طراحی شخصیت
1. وارد [Piskel](piskelapp.com) (اختیاری ولی ترجیح داده شده) میشید و شخصیت خود را میسازید.
2. بعد از بخش save گزینه save as .piskel را انتخاب میکنید
3. در بخش export ازش فایل zip یا png میگیرید
4. در ریپازیتوری تصاویر رو اپلود میکنید
5. تمام!
